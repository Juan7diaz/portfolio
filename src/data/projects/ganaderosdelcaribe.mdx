La finca El Ganaderos del caribe buscaba una solución para optimizar sus procesos y expandir su mercado. Como parte de un equipo de cuatro desarrolladores, tuve el privilegio de trabajar en la creación de "Ganaderos del Caribe", una plataforma web personalizada que transformó la gestión de esta finca ganadera.

## El Desafío

Necesitaba un sistema que no solo automatizara sus procesos internos, sino que también les permitiera vender sus productos más allá de su corregimiento. Nuestra tarea era crear una plataforma privada con una tienda en línea pública, todo mientras manteníamos altos estándares de seguridad y usabilidad.

## Nuestra Solución
Desarrollamos una aplicación web robusta y escalable, dividida en seis módulos principales:  

- 🐮 Registro de animales: Control detallado de cada res, desde su ID hasta su historial de salud.
- 📦 Control de inventario: Gestión eficiente de suministros y equipos de la finca.
- 🩺 Gestión de salud animal: Seguimiento preciso de tratamientos y vacunaciones.
- 📝 Registro de actividades: Monitoreo de todas las tareas diarias de la finca.
- 👥 Registro de trabajadores: Gestión completa del personal de El Ganado Feliz.
- 🛒 Tienda en línea pública: Venta directa de productos frescos de la finca a los clientes.

## Características Técnicas Destacadas

- **Front-end:** Utilizamos React para crear una interfaz de usuario moderna y responsiva. Material-Tailwind nos ayudó a lograr un diseño atractivo y consistente.
- **Back-end:** Implementamos una API RESTful con Express y TypeORM, conectada a una base de datos MSSQL.
- **Seguridad:** Implementamos autenticación con bcryptjs y JWT, asegurando la protección de los datos sensibles de la finca.
- **UX/UI:** Integramos React Beautiful DnD para interacciones fluidas y Notistack para notificaciones intuitivas.
- **Desarrollo y Despliegue:** Seguimos metodologías ágiles como Scrum y GitFlow, utilizando Azure DevOps para CI/CD.
