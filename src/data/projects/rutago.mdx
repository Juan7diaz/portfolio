¿Estás cansado de perderte en la ciudad? 🤔 **RutaGo** es la aplicación web funcional que estabas esperando. Desarrollada por un equipo de programadores web entusiastas, RutaGo te brinda una **solución innovadora** para descubrir las rutas de autobuses en **Santa Marta**. 🌴


## Características Principales

- 🌐 Frontend moderno con React y Chakra UI
- 🔒 Autenticación segura con bcryptjs, jsonwebtoken
- 📧 Recuperación de contraseña vía correo electrónico
- 🗺️ Visualización interactiva de rutas de autobús con Leaflet
- 🔐 Rutas públicas y privadas para usuarios autenticados
- 📡 Backend robusto con Express, TypeORM y PostgreSQL
- 👤 Vistas personalizadas para usuarios y administradores

## Tecnologías Utilizadas

- **Front-end:** *React, Chakra UI, Axios, Leaflet*
- **Back-end:** *bcryptjs, cors, dotenv, express, jsonwebtoken, typeorm, postgresql, nodemailer*

Nuestro equipo ( *Juan Díaz*, *Daniel Cogollos* y *Gian Astori* ) trabajó arduamente siguiendo una arquitectura MVC para entregar esta aplicación web escalable y lista para impresionar a reclutadores y empresas interesadas en soluciones de movilidad urbana. 🚀
