**TechMatch** es una plataforma de ecommerce especializada en componentes de computadora, diseñada para ofrecer una experiencia de **compra intuitiva y eficiente**. TechMatch permite a los usuarios encontrar fácilmente las piezas que necesitan para sus proyectos de PC.

## Características Principales

- 🔍 Buscador avanzado de componentes con filtros
- 🖥️ Visualización detallada de productos
- ⭐ Sistema de calificaciones y comentarios
- 🛒 Carrito de compras integrado
- 👤 Autenticación y registro de usuarios
- 💬 Comunicación directa vía WhatsApp

## Tecnologías Utilizadas

- **Front-end:** React, TypeScript, Zustand, Chakra UI, React Query
- **Back-end:** Spring Boot, Java, postgresql, mapstruct,
